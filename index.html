    Welcome to the UiPath Workshop¶

    Thank you for joining the UiPath Agentic Automation Practice Workshop!

    This Learner's Handbook is your practical guide for the hands-on exercises. Think of it less as a manual and more as a co-pilot: it will walk you through each step, explain what's happening and why, and help you build something real by the end of the session.

    Training Environment

    • Platform: cloud.uipath.com/tpenlabs
    • Tenant: AgenticPractice

    What You'll Build¶

    This workshop guides you through building real agentic automation solutions on the UiPath Platform. You'll work through two main use cases, each progressively adding intelligence and orchestration complexity:

    1. ServiceNow Incident Categorization Agent — Build an intelligent ServiceNow incident triage agent that reads incident descriptions for ServiceNow tickets and assigns the right category, subcategory, and on-duty assignee. Your first encounter with agentic AI in the wild.

    2. Invoice vs. Purchase Order Matching — a multi-actor automation that combines an AI agent with Robotic workflows, a human validation step, and API integrations. This is where agentic orchestration starts to look like real enterprise automation.

    Here's something worth holding onto as you work through these exercises:

    Specialized, domain-specific AI solutions consistently outperform general-purpose chatbots. Not because they're smarter, but because they're focused. And organizations that treat AI automation with the same engineering discipline they apply to software — version control, testing, deployment pipelines — consistently outperform those chasing the latest AI trend.

    That's what this workshop is about: building focused, well-engineered automations! The exercises apply software engineering discipline — structured process design, systematic testing, and incremental iteration — to AI automation projects.

    Key Concepts for this course¶

    • Agent Builder in Studio Web — Low-code environment for creating, designing, and deploying agents.
    • Maestro — UiPath's orchestration platform for coordinating humans, robots, and AI agents across long-running enterprise processes.
    • IXP (Intelligent eXtraction & Processing) — Document processing technology with confidence scoring for automated data extraction from unstructured documents.
    • Action Center and Apps— UiPath interface for human escalation workflows, exception handling, and manual review.
