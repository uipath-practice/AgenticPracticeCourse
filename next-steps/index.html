    Next Steps¶

    You've built a complete agentic orchestration. Here's what to do next.

    What You Accomplished¶

    By completing these exercises, you built a production-pattern automation that combines:

    Component Role
    Robots Structured, deterministic task execution
    AI Agents Intelligent decision-making and data processing
    Humans Judgment, validation, and exception handling
    APIs System integration and data exchange

    Immediate Next Steps¶

    1. Publish and Run Your Process¶

    Publish your completed process to your personal workspace, then run it as you would any other UiPath process.

    • Open Maestro in your workspace
    • Publish the process
    • Trigger a test run with sample data
    • Review the execution trace

    2. Explore Maestro Analytics¶

    After a few runs, Maestro analytics become available:

    • Transaction-level execution details
    • End-to-end process flow visualization
    • Bottleneck identification
    • Exception and escalation rates

    Open Maestro → Analytics to explore your process performance.

    Keep Iterating¶

    The real learning happens in experimentation:

    Prompt engineering - Modify your agent's system prompt - Test how changes affect categorization or matching decisions - Try different approaches for handling edge cases

    Test the boundaries - Feed your agent unusual or ambiguous inputs - See where it fails or produces unexpected results - Design for those failure modes

    Expand the process - Add a new document type to the matching process - Introduce a new escalation path - Connect to an additional system
