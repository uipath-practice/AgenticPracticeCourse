# 기술 지원 인시던트 분류하기

**티켓 라우팅을 위한 지능형 ServiceNow 인시던트 분류 에이전트 만들기**

## 개요

**UiPath Agent Builder**는 코딩 경험이 거의 없어도 지능형 자동화 솔루션을 개발할 수 있게 해 주는 핵심 도구입니다. 자동화 호출, API 활용, 다른 시스템과의 매끄러운 연동 등 다양한 작업을 수행하는 에이전트를 만들고, 설계하고, 배포할 수 있습니다.

Agent Builder는 로우코드 환경을 제공해 자동화 역량의 문턱을 낮춥니다. 덕분에 기술 전문가든 비전문가든 각자의 운영 요구에 딱 맞는 강력한 컨텍스트 기반 에이전트를 만들 수 있습니다.

이번 실습에서는 **UiPath Agent Builder**로 ServiceNow 인시던트를 자동으로 분류하고 라우팅하는 에이전트를 만듭니다. 의사 결정을 실제 데이터에 근거시키는 방법, 자동화된 테스트로 성능을 검증하는 방법, 에이전트를 실제 운영 시스템에 연결하는 방법을 배웁니다.

실습은 다음 순서로 진행됩니다.

- 먼저, 고정된 설명을 기반으로 인시던트를 분류하도록 에이전트를 가르치고, 샘플 입력으로 테스트하며 LLM의 동작을 관찰합니다.

- 다음으로, **Context Grounding**(컨텍스트 그라운딩)으로 결과를 다듬고 **Evaluations**(평가)로 성능을 검증합니다.

- 이어서, **Integration Service**로 에이전트를 **ServiceNow**에 연결해 "에이전트에게 손을 달아" 줍니다. 이제 에이전트가 인시던트 상세 정보를 조회하고 분석 결과에 따라 티켓을 업데이트할 수 있습니다.

- 마지막으로, **Escalations**(에스컬레이션)를 구성해 에이전트가 확신하지 못하는 케이스를 **Action Center**를 통해 사람에게 전달하도록 합니다.

실습을 마치면 프로덕션에 바로 투입할 수 있는, 완전하게 동작하는 분류 에이전트가 완성됩니다. 시작해 볼까요!
