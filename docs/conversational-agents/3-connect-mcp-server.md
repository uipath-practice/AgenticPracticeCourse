# Connecting an MCP Server

!!! tip "Here is our plan for this lesson:"

    1. Register an external MCP server in **Orchestrator**
    2. Expose the MCP server's weather tools to your agent
    3. Verify the agent can answer a weather question using the MCP tool

## Goal

You'll connect your agent to an external MCP server that provides weather data. This lesson introduces the MCP (Model Context Protocol) pattern in UiPath — how external tool servers are registered, governed, and made available to agents in Agent Builder.
