# 对话式智能体

**构建一个以 UiPath Security 文档为知识基础的对话式 AI 智能体，为它扩展工具，然后通过真实对话全面检验它的表现。**

## 概览

**Agent Builder** 让你可以创建实时回答问题的对话式智能体，它会依托有据可查的知识源和已连接的工具。这个练习将带你走完完整的生命周期：创建一个智能体，探索它可以使用的工具，并测试它在各种真实对话中的表现。

| 步骤 | 重点 |
| ---: | :--- |
| [**创建对话式智能体**](1-create-agent.md) | 使用 Context Grounding（上下文关联）构建一个以 UiPath Security 文档为知识基础的智能体 |
| [**了解可用工具**](2-review-tools.md) | 探索你的智能体可以使用的工具，以及它们如何处理数据 |
| [**测试对话**](3-testing-conversations.md) | 从不同角度发起对话，并评估回答的质量 |
