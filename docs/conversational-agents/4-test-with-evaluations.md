# Testing with Evaluations

!!! tip "Here is our plan for this lesson:"

    1. Create an evaluations dataset with representative questions and expected answers
    2. Run the evaluation against your conversational agent
    3. Review the results and understand how evaluation differs for conversational vs. autonomous agents

## Goal

You'll build an evaluations dataset and run it against your conversational agent. By the end of this lesson, you'll understand how the evaluations framework works for conversational agents, and how it compares to testing autonomous agents — where deterministic outputs are harder to define.
