# 대화형 에이전트

**UiPath Security 문서에 근거한 대화형 AI 에이전트를 만들고, 도구로 확장한 다음, 실제 대화로 성능을 검증해 봅니다.**

## 개요

**Agent Builder**를 사용하면 근거 있는 지식 소스와 연결된 도구를 활용해 실시간으로 질문에 답하는 대화형 에이전트를 만들 수 있습니다. 이 실습에서는 전체 라이프사이클을 다룹니다. 에이전트를 만들고, 에이전트가 사용할 수 있는 도구를 살펴보고, 다양한 실제 대화 속에서 얼마나 잘 버티는지 테스트합니다.

| 단계 | 내용 |
| ---: | :--- |
| [**대화형 에이전트 만들기**](1-create-agent.md) | 컨텍스트 그라운딩으로 UiPath Security 문서에 근거한 에이전트 만들기 |
| [**사용 가능한 도구 이해하기**](2-review-tools.md) | 에이전트가 사용할 수 있는 도구와 데이터 처리 방식 살펴보기 |
| [**대화 테스트하기**](3-testing-conversations.md) | 여러 각도에서 대화를 실행하고 응답 품질 평가하기 |
