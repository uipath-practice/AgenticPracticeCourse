# Step 2 — Configure a Robot

**Set up invoice PDF retrieval**

---

## Goal

Configure the robotic component to retrieve invoice PDF documents from the source system and pass them to the agent for IXP extraction.

## What the Robot Does

1. Connects to the document source (shared drive, email, DMS)
2. Identifies and retrieves unprocessed invoice PDFs
3. Passes the documents to the Maestro process for the next step
