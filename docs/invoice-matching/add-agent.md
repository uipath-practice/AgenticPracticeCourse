# Step 3 — Add an Agent

**Add AI-driven decision-making to your process**

---

## Goal

Integrate an AI agent into the Maestro process. The agent handles the parts of the workflow that require understanding, reasoning, and judgment — things a robot alone can't do.

## What the Agent Does

The invoice matching agent:

1. Receives invoice documents from the robot
2. Extracts key invoice data (vendor, amount, line items, PO references)
3. Queries the ERP system for the corresponding Purchase Order
4. Compares invoice data against the PO
5. Makes a matching decision: approve, reject, or escalate
