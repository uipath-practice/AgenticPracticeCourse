# Step 2 — Add a Robot

**Automate structured task execution**

---

## Goal

Add a robotic process automation component to handle the structured, rule-based parts of the workflow — tasks that are deterministic and don't require intelligent judgment.

## What the Robot Does

In the invoice matching process, the robot handles document retrieval and preparation:

- Connects to the source system
- Fetches invoice documents
- Prepares them for processing by subsequent workflow participants
