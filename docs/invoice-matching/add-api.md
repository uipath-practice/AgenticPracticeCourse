# Step 5 — Add API Calls

**Connect to external systems to complete the workflow**

---

## Goal

Complete the end-to-end process by adding API integrations that connect your Maestro process to external systems of record.

## What API Calls Enable

With API integration, your process can:

- Read Purchase Order data from the ERP system
- Post matching decisions back to the invoicing system
- Trigger downstream actions (payment approval, GL posting, notifications)
- Update status fields in connected systems
