    Creating a Conversational Agent¶

    Here is our plan for this lesson:

    1. Create a new conversational agent in Agent Builder
    2. Configure a system prompt scoped to UiPath Security topics
    3. Attach a UiPath Security documentation index as the context grounding source

    Goal¶

    You'll have a working conversational agent that answers questions about UiPath Security by drawing on official documentation. The agent won't guess or go off-topic — every answer is anchored to the grounded knowledge source you configure.
